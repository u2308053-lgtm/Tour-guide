# Tour-guide
A basic NLP chatbot that is used to help lost or eager tourists to find new destinations to explore or to find nearby stations or airports in case of emergencies

## Features

- Text tokenization
- Stopword removal
- Destination identification
- Intent classification
- Dataset-based response generation

## Provides information about:

- Destination state
- Nearest airport
- Railway station
- Category
- Popular attractions
- Accessibility
- Region
